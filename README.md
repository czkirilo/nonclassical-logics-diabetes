# Notebooks didáticos para Google Colab

Este pacote contém cinco notebooks sobre lógicas não clássicas aplicadas à base `Early Stage Diabetes Risk Prediction Dataset`.

## Arquivos

1. `01_logica_multivalorada_colab.ipynb`
2. `02_logica_fuzzy_colab.ipynb`
3. `03_logica_paracompleta_colab.ipynb`
4. `04_logica_paraconsistente_colab.ipynb`
5. `05_comparativo_geral_colab.ipynb`

## Como usar no Google Colab

1. Acesse https://colab.research.google.com/
2. Clique em `File > Upload notebook`
3. Envie o notebook desejado.
4. Execute a primeira célula e faça upload do arquivo `diabetes_data_upload.csv`.
5. Execute as células em ordem.

## Observação

Os notebooks foram escritos com foco didático. As implementações são heurísticas e transparentes,
pensadas para explicar a lógica de cada formalismo, e não para competir com modelos estatísticos
de machine learning.
